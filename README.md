## Running Locally


```sh-session
git clone https://github.com/Tom0Brien/tom
cd tom
```


Create a `.env` file similar to [`.env.example`](https://github.com/chronark/chronark.com/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
